# 医院白标页面信息提取模板 v1.0

> **目标**：精准、无遗漏地从合作医院官网提取结构化信息，用于生成白标页面
> **原则**：结构化抽取 > 自由总结，分层处理 > 整站爬取

---

## 📋 提取流程总览

```
Step 1: 官网结构扫描 → 识别关键页面 URL
Step 2: 分层提取内容 → 按页面类型使用不同 Prompt
Step 3: 图片资源收集 → 手动提取高清图片 URL
Step 4: 内容审核 → 检查完整性、删除联系方式
Step 5: 多语言翻译 → 使用术语对照表
Step 6: 生成白标组件 → WClinicMensContent.tsx
```

---

## 1️⃣ 官网结构扫描

访问目标官网，识别以下页面类型并记录 URL：

**必提取页面：**
- [ ] 首页 / 关于页（品牌信息）
- [ ] 治疗项目页（**逐个提取，最重要**）
- [ ] 医师团队页
- [ ] 价格/费用页
- [ ] 交通/访问页

**可选页面：**
- [ ] 设施设备页
- [ ] 常见问题（FAQ）
- [ ] 患者案例 / 治疗实绩
- [ ] 新闻 / 媒体报道

## 3️⃣ 图片资源收集

**手动提取方法**（浏览器开发者工具）：

1. 打开目标页面，按 F12 打开开发者工具
2. 切换到 **Network** 面板，筛选 `Img` 类型
3. 刷新页面，查看加载的图片资源
4. 右键图片 → Copy → Copy link address

**需要收集的图片：**

```markdown

- 医师1: URL

### 设备/设施照片
- 设备1: URL

### 地图图片
```

## 4️⃣ 内容审核清单

提取完成后，必须检查：

**完整性检查：**
- [ ] 至少 3 个治疗项目
- [ ] 至少 1 位医师信息
- [ ] 价格信息完整
- [ ] 交通信息准确

**合规性检查（白标页面强制要求）：**
- [ ] 删除所有电话号码
- [ ] 删除所有邮箱地址
- [ ] 删除 Line / WeChat / 社交媒体联系方式
- [ ] 删除"点击预约""电话咨询"等外部引导语
- [ ] 确认交通指南仅包含地理信息，无联系方式

**术语一致性检查：**
- [ ] 医疗术语使用 `medical-glossary.ts` 标准翻译
- [ ] 四种语言（ja / zh-CN / zh-TW / en）术语对齐
- [ ] 专业名词有日文原文标注

## 🔍 常见问题

### Q1: WebFetch 提取的内容不完整怎么办？
A: 分段提取。不要一次性抓取整个页面，而是按章节分别调用 WebFetch。

### Q2: 官网图片有 CORS 限制怎么办？
A:
1. 检查是否有 CDN 版本的 URL
2. 联系医院获取授权
3. 使用相似的库存图片替代（标注"示意图"）

### Q3: 医疗术语翻译不一致怎么办？
A: 严格使用 `lib/config/medical-glossary.ts`，所有新术语先添加到词汇表再翻译。

### Q4: 如何确保不遗漏重要信息？
A: 使用本模板的强约束 Prompt，AI 会按结构逐项填空，空字段会显示"未提及"。
